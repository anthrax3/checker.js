Checker

Checks if the field value already exists.

Getting Started

Download the production version or the development version.

In your web page:

<script src="libs/jquery/jquery.js"></script>
<script src="dist/Checker.min.js"></script>
<script>
    jQuery(function($) {
        $('input[name=login]').checker({
            check: true
        });
    });
</script>

<input name="login" />
